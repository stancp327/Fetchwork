# FetchWork UI Redesign — Wireframe Specs

## Shared: Top Nav
Logo • Dashboard • Freelancers • Jobs • Services • Create • Messages • Disputes • Admin • Profile

## 1. Dashboard
- Context Bar: Welcome + Mode Toggle (client/freelancer) + Primary CTA
- Onboarding Card: progress bar + next step CTA
- Stats Row: Active Jobs • Proposals • Earnings • Messages
- Recent Activity: timeline + empty-state tips
- Quick Actions: Post Job • Create Service
- Recommended Next: personalized suggestions
- Footer/Help: Support, Tutorials, Status

## 2. Discover Freelancers
- Header: title + value prop + CTA
- Search Bar: query + skills + location + budget + sort
- Filters Sidebar: Category • Rating • Availability
- Results Controls: count + saved search + grid/list toggle + sort
- Freelancer Cards Grid: avatar • badges • rating • headline • CTA
- Pagination / Load more

## 3. Browse Jobs
- Header: title + search + CTA
- Search Bar: keywords + category + experience + budget + sort
- Filters Sidebar: Remote/On-site • Location • Budget • Skills
- Result Controls: count + saved filters + sort + view toggle
- Job Cards List: title • budget • timeline • tags • apply CTA
- Pagination / Load more

## 4. Browse Services
- Header: title + search + CTA
- Search Bar: keywords + category + price + delivery + sort
- Filters Sidebar: Category • Price • Delivery • Rating
- Results Controls: count + saved searches + sort + grid/list
- Service Cards Grid: thumbnail • title • seller • starting at • CTA
- Pagination / Load more

## 5. Create Service (Wizard)
- Header: title + tips + Preview toggle
- Stepper: Details → Pricing → Media → Requirements → Review
- Form Panel: grouped sections (Details, Tags, Requirements, Packages)
- Live Preview Panel: service card + package summary
- Sticky Actions: Save Draft • Cancel • Continue/Publish

## 6. Profile
- Header: Profile + completion % + "Public Preview" toggle + Share/Profile URL
- Section Tabs: Overview • About • Skills • Portfolio • Rates • Verification • Settings
- Left Summary Card: Avatar + Name + Headline
- About Section: Bio (richtext) + Availability + Languages + CTA to Edit/Public Preview + "Add more"
- Skills & Expertise: skill chips + categories + years exp + "Add skill"
- Rates & Services: hourly rate + service packages shortcut + "Create service"
- Portfolio Highlights: 3-6 tiles + "Add project"
- Safety/Trust: Verify Email/Phone • ID badge • Payment setup status
- Sticky Actions: Save changes • Save draft • Cancel • Unsaved changes indicator

## 7. Messages
- Header: Messages + search + New message CTA
- Inbox: conversation list + filters (All/Unread)
- Chat Thread: history + attachments + composer
- Context Panel: job/service link
- Quick Actions: dispute • request payment • schedule

## Cross-Cutting
- ALL pages must be mobile responsive (current site is NOT mobile-friendly)
- Consistent nav across all pages
- Empty states with helpful CTAs
- Loading skeletons
